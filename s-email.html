Programming lab 2: Substituting emails with regular expressions

Submission deadline: Friday, September 21, 2012 (11:59 p.m.)

Introduction

The aim of this programming assignment is to write a Haskell program that searches and replaces an email with regular expressions.

Assignment (80%)

  • (70%) Write a Haskell program that searches an input file for an email and replaces it with the value of the email environment variable.
  • (10%) Annotate your Haskell source code with Haddock. Improve it with HLint.
  • (Optional) Package your Haskell program with Cabal.

Examples (Input and output)

If the email environment variable is set and an input file with an email is given, s-email replaces that email with the value of the email environment variable:

    $echo $email
    jpvillaisaza@gmail.com
    $ cat README.md
    s-email
    =======

    ---

    Juan Pedro Villa Isaza <jvillais@gmail.com>
    $ s-email README.md
    $ cat README.md
    s-email
    =======

    ---

    Juan Pedro Villa Isaza <jpvillaisaza@gmail.com>

If the email environment variable is set and an input file without an email is given, s-email does nothing:

    $ cat README.md
    s-email
    =======
    $ s-email README.md
    $ cat README.md
    s-email
    =======

If the email environment variable is not set, s-email displays an error message:

    $ cat README.md
    s-email
    =======

    ---

    Juan Pedro Villa Isaza <jvillais@gmail.com>
    $ s-email README.md
    the `email` environment variable is not set
    $ cat README.md
    s-email
    =======

    ---

    Juan Pedro Villa Isaza <jvillais@gmail.com>
